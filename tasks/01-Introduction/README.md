# التكليف 01

**الوصف:**
لديك أكثر من Code كلهم يقوموا بعمل نفس الوظيفة قم بكتابة تعليق
            Multiple Lines تشرح فيه باللغة الإنجليزية ما هو ال Code الذي لن يعمل
            مع ذكر السبب

**النقاط:**
- نوع العنصر هو h1
- خواص ال CSS هي كالتالي ويجب إضافتها بواسطة JavaScript

**الكود:**
```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta http-equiv="X-UA-Compatible" content="IE=edge" />
    <title>Learn JavaScript</title>
    <script>
      // Code One
      document.getElementById("el").style.color = "red";
    </script>
    <script>
      // Code Two
      window.onload = function () {
        document.getElementById("el").style.color = "red";
      };
    </script>
  </head>
  <body>
    <h1 id="el">Page Title</h1>
    <script>
      // Code Three
      document.getElementById("el").style.color = "red";
    </script>
  </body>
</html>
```
